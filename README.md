# VeriFlow Frontend

This is a Next.js 14 project for the VeriFlow Hackathon Application.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the outcome.

The page auto-updates as you edit the files in the `app` directory.
